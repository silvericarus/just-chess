Description

Just Chess, also stylized as JuSt Chess, is a web app that let you play chess in with another player. It uses HTML5 with vanilla Javascript, and the Bulma framework for design. In the current version of the game, known as 0.1.0 “The Turk”, the first “chess-playing” machine, that wasn’t really a machine, but was really discovered to be a person. Likewise, this version looks like you are playing with an automatic machine, but it’s a player. But maybe you already know that…

Instructions

If you want to create a new match:
  1. Click on the plus button below the match table.
  2. A new match with a random name will be generated, with state “waiting”.
  3. You click on the “ ” button on that match.
If you want to enter an existing match:
  1. Refresh to check if the match is still in “waiting” state.
  2. Click “ ”.
